# Online Shop

There is a one-page online shop with functions to hold products in cart

🔗 Live demo: [https://github.com/taras-stet-din0/mini-online-shop](link)

!Screenshot <img width="824" height="849" alt="image" src="https://github.com/user-attachments/assets/27e2fc4b-d9a2-40d4-8b20-fd50f68174d3" />

## Features
- product list loaded from JSON
-  shopping cart loads from localStorage

## Tech stack
- HTML5, CSS3 (Flexbox/Grid)
- JavaScript (ES6)

## How to run locally
1. Clone the repository: git clone [url]
2. Open index.html in your browser

## What I learned
Working with the DOM, handling events, and storing
data in localStorage.
